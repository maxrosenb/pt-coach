# Audio Files

Place native Portuguese audio recordings here with the following naming convention:
- `{phrase_id}.mp3`

Example:
- `1.mp3` - Audio for "Bom dia"
- `2.mp3` - Audio for "Como vai?"
- etc.

For MVP, you can use:
1. Text-to-speech services (Google TTS, Amazon Polly, etc.)
2. Record native speakers
3. Use existing Portuguese audio resources

Recommended TTS: Google Translate TTS with Portuguese (Portugal) voice for accurate pronunciation.
